# 🤖 Paczka Skilli do Claude Code — od proflowsync

5 gotowych skilli, które zamieniają Claude Code w Twój marketingowy sztab. Wrzucasz, wpisujesz `/nazwa` i masz gotową robotę — treści, copy, SEO, maile, prompty.

## ⚡ Instalacja (30 sekund)

1. Skopiuj foldery skilli do katalogu skilli Claude Code:
   - **Globalnie (wszystkie projekty):** `~/.claude/skills/`
   - **Dla jednego projektu:** `.claude/skills/` w repo
2. Zrestartuj Claude Code (albo `/reload`)
3. Wpisz `/` i zobaczysz nowe skille na liście

```
~/.claude/skills/
├── content-machine/SKILL.md
├── copy-pro/SKILL.md
├── seo-optimizer/SKILL.md
├── email-flow/SKILL.md
└── prompt-master/SKILL.md
```

## 📦 Co jest w paczce

| Skill | Co robi | Wywołanie |
|-------|---------|-----------|
| **content-machine** | Posty, hooki, scenariusze pod social media | `/content-machine` |
| **copy-pro** | Copywriting sprzedażowy: oferty, CTA, opisy | `/copy-pro` |
| **seo-optimizer** | Słowa kluczowe, meta, struktura artykułu | `/seo-optimizer` |
| **email-flow** | Sekwencje maili, newslettery, follow-upy | `/email-flow` |
| **prompt-master** | Optymalizacja i wzmacnianie promptów AI | `/prompt-master` |

## 🎯 Jak z tego wycisnąć maksimum

- Łącz skille: `/content-machine` → pomysł na short, potem `/copy-pro` → opis pod niego
- Każdy skill pyta o kontekst (nisza, ton, cel) — im konkretniej, tym lepszy efekt
- Podmieniaj `[ZMIENNE]` w promptach na swoje dane
